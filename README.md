Project Name  : Digital Tools Platform

Description   :  
    Digital Tools Platform is a web-based application where users can explore and access various useful digital tools in one place. It provides       an easy and user-friendly interface to browse tools, view details, and manage selections efficiently.

Technologies Used :   
    React.js
    Tailwind CSS
    DaisyUI
    JavaScript (ES6+)
    React-Toastify (NPM Package)
    JSON (for product data)

Key Features  :  
    Users can browse and explore different digital tools بسهولة.
    Add-to-cart functionality with real-time notifications using Toast.  
    Fast and responsive UI with clean design using Tailwind CSS & DaisyUI.
